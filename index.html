知识库

权威医学信息，帮助您全面了解白塞病

什么是白塞病？

白塞病（Behcet's Disease，BD），又称贝赫切特综合征，是一种原因不明的以血管炎为基础病理改变的慢性、复发性、多系统疾病。主要表现为反复发作的口腔溃疡、生殖器溃疡、眼部病变和皮肤损害，严重时可累及神经系统、消化系统和大血管。该病多见于\“丝绸之路\”沿线国家，在我国及东亚地区具有较高发病率，男女均可患病，好发于20-40岁青壮年人群。

病因：遗传与环境的共同作用

目前认为白塞病的发病与遗传易感性和环境因素共同作用有关。遗传方面，HLA-B51基因与白塞病具有显著相关性，携带该基因的人群患病风险较高。环境因素方面，感染（尤其是链球菌感染）、免疫系统异常激活被认为是重要诱发因素。此外，氧化应激及免疫调节异常也在发病机制中扮演重要角色。

主要症状

👄

口腔溃疡

反复发作的口腔黏膜溃疡，疼痛明显，是最常见且往往最早出现的症状，每年发作≥3次为特征性表现。

👁️

眼部病变

葡萄膜炎（前/后）、视网膜炎，可导致视力下降甚至失明，是最严重的并发症之一，需及时治疗。

🩺

生殖器溃疡

外生殖器出现溃疡，疼痛，愈后多留有瘢痕，常为诊断提供重要依据。

🦷

皮肤表现

结节性红斑、假性毛囊炎、丘疹脓疱性病变等，皮肤针刺反应（Pathergy test）阳性具有诊断意义。

🦴

关节炎

非破坏性、非对称性关节炎，常累及膝、踝等大关节，可伴有关节肿痛，一般不造成关节畸形。

🧠

神经系统病变

神经白塞病（Neuro-Behcet）可出现头痛、颅内压增高、脑干损害等，是预后较差的并发症。

🩸

血管病变

静脉血栓形成（深静脉血栓）、动脉瘤，大血管受累可危及生命，需积极治疗。

🍽️

消化道病变

肠白塞病可出现腹痛、腹泻、消化道溃疡，需与炎症性肠病鉴别，严重时可导致穿孔。

用药须知

白塞病目前无特效根治药物，治疗目标是控制急性发作、预防复发及脏器损害。以下药物信息仅供参考，具体用药方案须在专科医生指导下制定，切勿自行增减药量。

💊 激素类药物 / Corticosteroids
🔴 泼尼松 / Prednisone
用途：控制炎症急性发作，用于眼部病变、血管炎、神经系统受累等。减轻炎症反应，改善急性期症状。
⚠ 长期使用需监测血糖、骨密度，注意感染风险，不可突然停药。
🔴 地塞米松 / Dexamethasone
用途：强效糖皮质激素，常用于眼部局部注射或急性重症发作，抗炎效果强。
⚠ 作用强效，需在医生指导下使用，避免长期全身使用带来的副作用。
💉 免疫抑制剂 / Immunosuppressants
🔵 环磷酰胺 / Cyclophosphamide
用途：用于严重眼部病变、神经白塞病、重度血管炎，抑制免疫系统过度激活。
⚠ 骨髓抑制、出血性膀胱炎风险，需定期监测血常规及尿常规。
🔵 甲氨蝶呤 / Methotrexate
用途：用于皮肤黏膜病变、关节炎，减少溃疡发作频率，维持缓解期。
⚠ 肝功能损伤风险，禁止饮酒，同服叶酸可减少副作用，每周仅服一次。
🔵 硫唑嘌呤 / Azathioprine
用途：维持期免疫抑制，常用于眼部及系统性白塞病的长期管理，防止复发。
⚠ 骨髓抑制，需定期复查血常规，与别嘌醇合用需减量。
🔵 秋水仙碱 / Colchicine
用途：用于口腔溃疡、结节性红斑、关节炎的预防和治疗，减少发作频率。
⚠ 胃肠道刺激，肾功能不全者需调整剂量，过量可致毒性反应。
🧬 生物制剂 / Biologics
🟢 英夫利西单抗 / Infliximab
用途：TNF-α拮抗剂，用于难治性眼部白塞病、神经白塞病及其他重症表现，效果显著。
⚠ 需排除活动性结核、严重感染，使用前须筛查TB，价格较高，需医院输液。

国际白塞病研究组诊断标准（ICBD）

目前广泛采用的是2014年修订的国际白塞病研究组（ICBD）评分诊断标准，总分≥4分可诊断为白塞病。

症状 / Symptom 分值 / Score 说明 / Description
口腔溃疡2分反复发作（≥3次/年），边界清楚，疼痛性口腔黏膜溃疡
生殖器溃疡2分反复发作的外阴溃疡，愈后常留有瘢痕
眼部病变2分前葡萄膜炎、后葡萄膜炎、全葡萄膜炎或视网膜血管炎
皮肤病变1分结节性红斑、假性毛囊炎或丘疹脓疱性病变
神经系统病变1分中枢神经系统受累，排除其他原因
血管病变1分动脉或静脉血栓、静脉曲张、动脉瘤
针刺反应阳性1分皮肤针刺后24-48小时出现丘疹或脓疱

注：总分≥4分可诊断为白塞病。诊断需排除其他类似疾病（如炎症性肠病、系统性红斑狼疮、复发性口疮等）。

白塞病预后

白塞病的预后因个体差异而不同。总体而言，白塞病是慢性复发性疾病，多数患者经过规范治疗可以维持较好的生活质量。眼部受累是影响预后的重要因素，未经及时治疗可导致视力永久损害甚至失明。神经系统及大血管受累者预后相对较差，需要积极治疗。约1/3患者病情可随时间推移而自然缓解，男性患者通常比女性症状更严重。

✅

预后良好因素

女性患者、发病年龄较晚、仅有皮肤黏膜受累、早期诊断并规范治疗。

⚠️

预后不良因素

年轻男性患者、早期出现眼部病变、神经系统或大血管受累、疾病活动度高。

🔄

规律随访建议

每3-6个月定期复查，监测眼部情况，及时调整治疗方案，遵医嘱规律用药不擅自停药。

🌿

生活方式管理

戒烟戒酒、避免过度劳累、均衡饮食、适度运动、保持良好心态，积极参与病友支持群体。

💡 就诊贴士

  • 提前准备好所有历次检查报告、化验单、用药记录，以供医生参考。
  • 建议通过医院官方App或网络挂号平台提前预约专家号，避免现场久等。
  • 就诊时如实描述症状发作频率、持续时间及加重/缓解因素。
  • 首次就诊建议选择风湿免疫科，如有眼部症状同时挂眼科。
  • 记录症状日记，包括口腔溃疡照片（手机拍摄即可），帮助医生判断病情活动度。

关于我们

由学生主导，为患者服务的公益性信息互助平台

📋 项目背景

白塞病是一种相对罕见却严重影响生活质量的自身免疫性疾病。然而，大多数患者在确诊前需要辗转多家医院，平均确诊时间长达数年。缺乏权威的中文科普资源、就医渠道不清晰、患者间缺少交流平台，是患者群体面临的主要困境。

基于对70余位白塞病患者的深度问卷调研，我们发现以上问题普遍存在。为此，一支由在校大学生组成的团队发起了\“白塞病联盟\”项目，立志用力所能及的方式改善这一现状。

🎯 项目使命

让每一位白塞病患者都能获得：

  • 准确、权威的疾病认知
  • 便捷的就医指引与专家资源
  • 温暖的病友社群支持
  • 持续更新的医学进展信息

👥 项目团队

我们是一支充满热情的在校学生团队，成员涵盖医学、设计、技术等多个专业，共同致力于推动白塞病患者信息获取的平等化与便利化。我们与白塞联盟合作，确保内容的专业性与准确性。

🤝 支持机构

本项目受到白塞联盟的指导与支持。白塞联盟致力于推动国内白塞病患者群体的组织化支持与医患沟通，是本项目最重要的合作伙伴。

⚠️ 免责声明

本网站所有内容仅供参考，不构成医疗建议。白塞病的诊断与治疗应在专业医生的指导下进行。如有任何医疗疑问，请及时就诊。
